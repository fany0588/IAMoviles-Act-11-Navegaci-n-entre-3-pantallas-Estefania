## Mi promt Navegación 3 paginas 
Lenguaje Dart flutter, Nivel principiante, ejemplo de navegación entre 3 páginas utilizando rutas nombradas, desde main llamar a página 1, En la página mostrar el appbar el titulo "Inicio" con el color de texto blanco y el fondo rosa fuerte lindo y en body, en una columna mostrar texto (Página Inicial) mas un botón color rosita claro con letra negra para ir a la segunda página, en la segunda página mostrar el appbar "Segunda Página" con el texto color blanco y fondo azul y en body mostrar una imagen desde la red mas un botón color azul clarito letra blanca para ir a la tercera página, en la tercera página en appbar mostrar titulo "Tercera Página" con texto color blanco y fondo rojo y en body mostrar un container de 200 x 200 rojo clarito. Debe ser atractivo, fácil navegación y mostrar todo el código en un solo archivo.
